# rust-tetris

CUI style tetris.

This is a [c_tetris](https://github.com/ABC10946/c_tetris) clone made with Rust.

# build and run

```
$ cargo run
```

# control

## 操作方法

| key       | control          |
| --------- | ---------------- |
| arrow key | move tetrimino   |
| R         | rotate tetrimino |
| Q         | close program    |
